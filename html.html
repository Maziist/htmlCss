HTML (HyperText Markup Language)

HTML signifie « HyperText Markup Language » qu'on peut traduire par « langage de balises pour l'hypertexte ». Il est utilisé afin de créer et de représenter le contenu d'une page web et sa structure. D'autres technologies sont utilisées avec HTML pour décrire la présentation d'une page (CSS) et/ou ses fonctionnalités interactives (JavaScript).

Structure de base d'un document HTML

Un document HTML commence par la déclaration , suivie par une structure de balises qui forment le squelette de la page :

  • < !DOCTYPE html : Indique au navigateur que le document est en HTML5.
  • < html> : La balise racine de la page.
  • < head> : Contient des meta-informations, le titre de la page, et les liens vers les ressources externes (comme CSS et JavaScript).
  • < body> : Contient le contenu visible de la page, tel que les titres, les paragraphes, les images, etc.

Les Balises HTML

Les balises HTML sont généralement en paires : une balise d'ouverture < tag> et une balise de fermeture < /tag>. Voici quelques balises couramment utilisées :

  • < h1> à < h6> : Titres de différents niveaux, < h1> étant le plus important.
  • < p> : Paragraphe.
  • < a> : Lien hypertexte, utilisé pour créer des liens entre les pages.
  • < img> : Image, avec des attributs comme src pour la source de l'image et alt pour le texte alternatif.
  • < ul> et < li> : Liste non ordonnée et ses éléments.
  • < ol> et < li> : Liste ordonnée et ses éléments.
  • < div> : Division ou section, utilisée pour regrouper des éléments et appliquer des styles CSS.
  • < span> : Utilisée pour appliquer des styles à une partie de texte sans créer de bloc séparé.

Attributs HTML

Les balises HTML peuvent avoir des attributs qui fournissent des informations supplémentaires. Par exemple :

  • href : Spécifie l'URL du lien.
  • target="_blank" : Ouvre le lien dans un nouvel onglet.

Les Formulaires

Les formulaires permettent aux utilisateurs d'entrer des données. Voici un exemple de formulaire simple :

  • < form> : Définit le formulaire.
  • action : URL où les données du formulaire seront envoyées.
  • method : Méthode de soumission (GET ou POST).
  • < label> : Étiquette pour un champ de formulaire.
  • < input> : Champ de saisie.

Conclusion

HTML est essentiel pour créer la structure et le contenu des pages web. Bien que le HTML seul permette de créer des pages web,
il est souvent utilisé en conjonction avec CSS (Cascading Style Sheets) pour le design et JavaScript pour l'interactivité.
