CSS (Cascading Style Sheets)

CSS permet de créer des pages web à l'apparence soignée. Cet article vous propose de lever le voile en expliquant ce qu'est CSS ; un exemple simple en présentera la syntaxe, puis quelques termes clés du langage seront introduits.

Structure de base d'un fichier CSS

Un fichier CSS est composé de règles de style qui ciblent des éléments HTML. Chaque règle de style est composée d'un sélecteur et d'un bloc de déclaration :

Exemple :

body : Le sélecteur qui cible l'élément < body>.

background-color et font-family : Les propriétés de style.

#f0f0f0 et Arial, sans-serif : Les valeurs des propriétés.

Méthodes d'Intégration du CSS

Il existe trois façons d'ajouter du CSS à un document HTML :

CSS en ligne :
Utilisation de l'attribut style directement dans les éléments HTML.

*

CSS interne :
Utilisation d'un bloc style> dans la section head> du document HTML.

CSS externe :
Utilisation d'un fichier CSS séparé. Le fichier CSS est lié dans la section < head> du document HTML.

Sélecteurs CSS

Les sélecteurs permettent de cibler les éléments HTML auxquels les styles seront appliqués. Quelques sélecteurs courants :

Sélecteurs de type : Ciblent tous les éléments d'un type donné.

Sélecteurs de classe : Ciblent les éléments avec une classe spécifique.

Et dans HTML :

Sélecteurs d'ID : Ciblent un élément avec un ID spécifique.

Et dans HTML :

Sélecteurs d'attribut : Ciblent les éléments avec un attribut spécifique.
